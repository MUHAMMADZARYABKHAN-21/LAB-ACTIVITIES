# LAB-ACTIVITIES

This repository contains my Object-Oriented Programming (OOP) lab activities and practice programs written in C++.

## About
These programs are created as part of my OOP coursework to practice and understand important programming concepts such as:

- Classes and Objects
- Constructors and Destructors
- Inheritance
- Polymorphism
- Function Overloading
- Function Overriding
- Virtual Functions
- Pure Virtual Functions
- Abstract Classes
- Dynamic Binding
- Static Binding
- File Handling
- Exception Handling
